# 🖋️ Portfólio Jesse Artes

Este é um projeto de portfólio de alta performance desenvolvido para o artista **Jesse Artes**, focado em exibir trabalhos de realismo e transformações artísticas.

## 🎨 Sobre o Artista
Jesse é especialista em transformar peles através do **Realismo em Preto e Cinza**. Seu trabalho destaca-se pela precisão nos detalhes e pela capacidade técnica de realizar **coberturas (cover-ups)** e **revitalizações**, devolvendo a estética e o orgulho aos clientes com tatuagens antigas.

## 🚀 Funcionalidades do Site
- **Galeria Imersiva:** Visualização em alta definição dos trabalhos em Black & Grey.
- **Slider Antes/Depois:** Demonstração interativa do processo de cobertura e revitalização.
- **Agendamento Direto:** Integração com WhatsApp para consultoria de orçamentos.
- **Design Responsivo:** Otimizado para visualização em dispositivos móveis.

### 🛠️ Como essas três interagem no seu Portfólio?

No seu link do GitHub Pages, eu percebi que você estruturou bem a semântica. Para um desenvolvedor **Full Stack** em formação como você, entender essa separação é crucial:

1.  **O HTML** carrega o conteúdo (essencial para o Google achar o James Tattoo - SEO).
2.  **O CSS** cria a atmosfera de um estúdio de tatuagem (experiência do usuário - UX).
3.  **O JavaScript** facilita o contato do cliente com o artista (conversão/funcionalidade).
